# Regulatory Traceability Matrix

The consolidated regulatory traceability matrix is maintained in the documentation site:

**Primary location:** `docs-site/docs/regulatory-traceability/traceability-matrix.md`

This file provides the single auditable mapping of all extracted business rules to their applicable regulatory requirements (FSA FFFS 2014:5, PSD2, GDPR, AML/KYC, DORA, EBA Outsourcing Guidelines, EU Consumer Credit Directive).

To view the rendered version, build and serve the docs site:

```bash
cd docs-site && npm run build && npm run serve
```
